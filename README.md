# Fresh Start - Chrome Extension

Простое расширение для Chrome, которое очищает localStorage текущей страницы, перезагружает её и автоматически заполняет форму авторизации.

## Функциональность

- **Очистка localStorage**: Очищает все данные localStorage текущей страницы
- **Автоперезагрузка**: Автоматически перезагружает страницу после очистки
- **Автозаполнение формы**: Находит поля логина и пароля и заполняет их заданными значениями
- **Автологин**: Автоматически нажимает кнопку Далее для входа в систему
- **Настраиваемые параметры**: Все названия полей, значения и автологин можно настроить в настройках расширения

## Установка

1. Откройте Chrome и перейдите в `chrome://extensions/`
2. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
3. Нажмите "Загрузить распакованное расширение" (Load unpacked)
4. Выберите папку `storage-clearer`
5. Расширение будет установлено и появится в списке расширений

## Использование

1. Откройте страницу, на которой нужно выполнить действия
2. Нажмите на иконку расширения в панели инструментов Chrome
3. Нажмите кнопку "Выполнить"
4. Страница автоматически:
   - Очистит localStorage
   - Перезагрузится
   - Заполнит поля логина и пароля

## Настройки

1. Нажмите на иконку расширения
2. Нажмите кнопку "Настройки"
3. Задайте параметры:
   - **Название поля Логин**: Текст, по которому будет искаться поле логина (по умолчанию: "Логин")
   - **Название поля Пароль**: Текст, по которому будет искаться поле пароля (по умолчанию: "Пароль")
   - **Значение для поля Логин**: Значение, которое будет вставлено в поле логина (по умолчанию: "admin")
   - **Значение для поля Пароль**: Значение, которое будет вставлено в поле пароля (по умолчанию: "admin123")
4. Нажмите "Сохранить настройки"

## Технические детали

- Использует Manifest V3
- Сохраняет настройки в `chrome.storage.sync`
- Использует `sessionStorage` для передачи данных между перезагрузками
- Ищет поля по различным атрибутам: name, id, placeholder, aria-label, label
